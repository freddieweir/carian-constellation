üöÄ Deployment Steps

  1. Verify Ephemeral Infrastructure:
    cd ../ephemeral && terraform output
    Ensure EKS cluster is running and accessible
  2. Configure Variables:
    cp terraform.tfvars.example terraform.tfvars
    vim terraform.tfvars
    Fill in: domain, email, Grafana password, Terraform state bucket
  3. Initialize Terraform:
    terraform init
    terraform validate
  4. Plan Deployment:
    terraform plan
    Review all resources to be created (~40-50 resources)
  5. Deploy Infrastructure:
    terraform apply
    Deployment takes ~10-15 minutes
  6. Verify Deployment:
    kubectl get pods --all-namespaces
    helm list --all-namespaces
    kubectl get clusterissuer
    kubectl get clustersecretstore
  7. Access Grafana:
    terraform output grafana_url
    Login with username admin and your configured password
  8. Test TLS Certificates:
    Deploy a test Ingress with cert-manager annotations
    Verify certificate is issued (may take 1-2 minutes)
  9. Create Secrets in AWS:
    aws secretsmanager create-secret --name carian-constellation/app/db-pass --secret-string "password"
    Test with ExternalSecret resource
  10. Deploy Applications:
    Ready to deploy your workloads!
    cd ../../kubernetes && kubectl apply -f namespaces/
